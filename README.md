# 🌈 Rainbow Starter

Fullstack starter template with:

- Node.js + Express
- MongoDB
- JWT Authentication
- Protected routes
- React + React Query
- TypeScript

## 🚀 Features

- User login
- JWT authentication
- Auth middleware
- Protected `/me` route
- Clean architecture

## ⚙️ Setup

### Backend

```bash
cd backend
npm install
npm run dev

### Frontend
npm install
npm run dev

### EsLint e Prettier
npm run lint -> lanciare EsLint
npm run lint:fix -> Fixare i problemi
npm run format -> Per formattare tutto
```
